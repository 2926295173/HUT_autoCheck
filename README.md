## HUT_autoCheck

**湖南工业大学完美校园健康打卡**
- 随机温度(36.2℃-36.8℃)🌡，随机经纬度🌍
- 多人打卡👨‍👩‍👧‍👧，一人微信通知全部打卡结果💬
- 校内打卡:6:30🕑
- 基于Github Actions，完全解放服务器✔
